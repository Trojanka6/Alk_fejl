# Alk_fejl
Akalmazások fejlesztése beadandó

01_Projektötlet

Tantárgyak felvétele / mini Neptun

Célunk egy olyan alkalmazás elkészítése, mely lehetővé teszi a tanulóknak, hogy tantárgyaikat gyorsan és egyszerűen fel tudják venni. 
Nem minden tanuló jogosult minden tantárgyat felvenni, ezt a neptun kódjuk alapján lehet ellenőrizni bejelentkezés után.
Betelt kurzusok esetén üzenet a felhasználónak.
